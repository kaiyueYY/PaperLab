# PaperLab

PaperLab is an AI-powered literature research workspace for academic writing.

## Core Modules

- `Literature Review`: topic-driven deep research flow, paper extraction, review generation, export
- `Literature Library`: local paper management, grouping/sorting, metadata detail modal, source/full-text links

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Configure environment variables in `.env` or `.env.local`
3. Start development server:
   `npm run dev`
